# Accounts - Sistema de Validación de Identidad Biométrica

Sistema de autenticación multi-método (Google, Face Recognition, RFID) para aplicaciones de Punto Indigo.

## Desarrollo Local

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## Build

```bash
npm run build
npm start
```
